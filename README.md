# CSS811_Slink

Using CSS811-Breakoutboard: CJMCU-811.

Output of CSS811 is CO2, TVOC and temperature   mapped to BME680-SLINK-UDP-Transmission, in order to 
get these readings in [FHEM](https://forum.fhem.de/index.php).
